# Live Fashion Link Hub

Landing page mobile-first para organizar produtos indicados em uma live de moda.

## Onde editar

- Informações da influenciadora, live, data, horário e Instagram: `data/live.ts`
- Lista de produtos, categorias, imagens e links de compra: `data/live.ts`
- Imagens dos produtos: `public/products`
- Componentes da página: `components`

## Rodar localmente

```bash
npm install
npm run dev
```

Depois acesse `http://localhost:3000`.

## Deploy na Vercel

Suba este projeto em um repositório Git e importe na Vercel. O preset de Next.js será detectado automaticamente.
